

# 📘 JavaScript Problem Solving – Chapter Wise Practice

This repository contains my full **JavaScript Problem-Solving Practice**, organized chapter-by-chapter.  
Anyone learning JavaScript (Beginner → Intermediate) can easily understand and follow along.

## 📂 Chapter Overview

নিচে প্রতিটি ফোল্ডারে আলাদা করে চ্যাপ্টার ভিত্তিক practice এবং solutions রাখা আছে:

### ✔ Chapter 1 – Variable  
Basic variable concepts, data storing, simple math based problem-solving.

### ✔ Chapter 5 – If / Else  
Condition, decision making, logical flow control based problems.

### ✔ Chapter 6 – Array  
Array handling, indexing, looping through arrays, and common array operations.

### ✔ Chapter 7 – Loop  
For loop, While loop এবং Loop দিয়ে বিভিন্ন logic build practice.

### ✔ Chapter 8 – Object  
Object properties, accessing values, modification, এবং object-based problems.

### ✔ Chapter 9 – Function  
Function creation, parameters, return values, reusable logic writing.

### ✔ Chapter 10 – Logic Build  
Pure logic-based problems to improve thinking ability & problem-solving skills.

### ✔ Chapter 11 – ES6 Modern JavaScript  
Arrow function, template string, spread, destructuring, let/const & modern concepts.
